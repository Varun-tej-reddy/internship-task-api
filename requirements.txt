## Scalability Note

This project follows a modular architecture with separate layers for models, routes, schemas, and utilities. 
To scale:

- Move to Dockerized deployment
- Add Redis caching for task queries
- Implement load balancing with Nginx
- Separate auth and task services into microservices
- Use async DB drivers for high concurrency
